### traceroute

追踪 IP（即跟踪数据包的路由路径）
```bash
traceroute example.com
```

| 命令                | 作用          | 适用系统      |
|---------------------|-------------|--------------|
| `traceroute IP/域名` | 追踪数据包路由 | Linux/macOS  |
| `tracert IP/域名`   | 追踪数据包路由 | Windows      |
| `mtr IP/域名`       | 实时路由跟踪  | Linux/macOS  |
| `ping IP/域名`      | 测试 IP 是否可达 | Linux/Windows |
