### 基本语法

``` bash
|、-、:之间的多余空格会被忽略，不影响布局。
默认标题栏居中对齐，内容居左对齐。
    -:表示内容和标题栏居右对齐
    :-表示内容和标题栏居左对齐
    :-:表示内容和标题栏居中对齐
内容和|之间的多余空格会被忽略，每行第一个|和最后一个|可以省略，-的数量至少一个。
```

### 简单表格

``` bash
表头| 表头
-|-
单元格|单元格
单元格|单元格
单元格|单元格
```

### 对齐方式

``` bash
-: 设置内容和标题栏居右对齐。
:- 设置内容和标题栏居左对齐。
:-: 设置内容和标题栏居中对齐。
```

``` bash
| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```
